Welcome to Artsy

Artsy is a new programming language that aims to educate younger students or simply those new to codeing on how to program in industry-standard programming languages. Our product provides an accessible web-based programming language for middle school students, with the major goal of creating a new language that students can start learning in middle school and progress to such a point that they become ready to program in other industry-standard languages. Moreover, the hosted programming language acts as a low-barrier learning platform for the students to grasp computer science skills for the future.

Artsy is the perfect learning tool for anyone looking to get a jump start on their coding education. With our language, students can learn the basics of programming in an intuitive, easy to understand way. Our language also acts as a transpiler, meaning it can convert into other commonly used programming languages. Artsy also provides an interactive web-based editor so that students can test out their code and programs from the comfort of their own homes. As a result, Artsy offers a comprehensive package for students to learn programming and develop their skills for the future..
