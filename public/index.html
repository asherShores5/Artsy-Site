Welcome to Artsy

Artsy is a web-based programming language designed to teach middle school students and beginners how to code using industry-standard languages. Our goal is to create a low-barrier learning platform that allows students to progress from basic programming skills to more advanced languages. With Artsy, students can develop computer science skills at an early age and be better prepared for the future.

Artsy is an intuitive programming language that acts as a transpiler, converting into commonly used languages. It's perfect for beginners and offers an interactive web-based editor for testing code from home. Artsy provides a comprehensive package for students to learn programming and develop skills for the future.
